# **App Name**: GuardiaSwap

## Core Features:

- Shift Calendar: Interactive calendar view displaying scheduled shifts.
- Shift Posting: Nurses can post shifts they want to swap, including location, time, and contact information.
- Shift Browsing: Browse available shifts posted by other nurses, filtered by date and location.
- Real-time Updates: Ensure shifts update in real-time via Firestore database, avoiding delays and conflicts.
- Contact via WhatsApp: Contact the nurse who posted the shift directly through WhatsApp.

## Style Guidelines:

- Primary color: Indigo (#4F46E5) to convey trust and reliability.
- Background color: Light gray (#F9FAFB) for a clean and modern feel.
- Accent color: Teal (#008080) to highlight important actions.
- Body font: 'Inter', a sans-serif font for readability.
- Headline font: 'Inter', a sans-serif font for readability.
- Use clear, modern icons to represent shift locations and times.
- Subtle transitions and animations when interacting with the calendar and shifts.
